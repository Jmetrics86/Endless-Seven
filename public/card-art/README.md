# Card art

Drop card art images here to use as textures on the cards.

## File naming

Name each file **exactly** as the card name in the game (from the card list), with a supported image extension.

**Examples:**
- `The Almighty.png`
- `Saint Michael.jpg`
- `Wild Wolf.webp`
- `Fallen One.png`

Spaces and punctuation must match. Supported formats: `.png`, `.jpg`, `.jpeg`, `.webp`.

## Card names (for reference)

**Light / Celestial:** The Spinner, The Allotter, Prophet, The Inevitable, Saint Michael, Martyr, The Almighty, Archangel, Cherubim, Fallen One, Herald, Nephilim, Seraphim, Thrones.

**Lycan:** Alpha, Beta, Omega, Sentinel, Delta, Luna, Wild Wolf.

**Darkness / Daemon / Vampyre:** Death, Famine, Pestilence, War, Lilith, Hades, The Destroyer, Wrath, Pride, Greed, Sloth, Envy, Lust, Gluttony, Lord, Duke, Elder, Noble, Regent, Baron, Fledgeling.

## Usage

The game will look for an image in this folder matching each card’s name. If a file is found, it will be applied as the card’s face texture. If not, the card uses the default generated face.
